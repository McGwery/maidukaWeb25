# MaiDuka Landing Page - Features & Documentation

## Overview
A professional, modern landing page designed for MaiDuka - a revolutionary multi-shop management system. The landing page uses an emerald color scheme and follows best practices for B2B SaaS marketing.

## Design System

### Color Palette
- **Primary Color**: Emerald (emerald-500 to emerald-600)
- **Secondary Color**: Teal (teal-500 to teal-600)
- **Accent Colors**: White, Gray scales
- **Dark Mode**: Fully supported with emerald theme

### Typography
- **Headings**: Bold, extrabold font weights
- **Body**: Regular font weights with proper line heights
- **CTAs**: Semibold with proper sizing

## Responsive Design

### Breakpoints
- **Mobile**: < 768px (Single column, stacked layout)
- **Tablet**: 768px - 1024px (2-column grids)
- **Desktop**: > 1024px (Full multi-column layouts)

### Mobile Optimizations
- Hamburger menu for navigation
- Stacked CTAs on mobile
- Single-column feature cards
- Adjusted typography sizes
- Touch-friendly buttons (py-4 padding)

## Accessibility Features

- Semantic HTML structure
- Proper heading hierarchy (h1, h2, h3)
- Alt text for icons (via sr-only spans)
- Keyboard navigation support
- High contrast ratios
- Focus states on interactive elements

## Performance Optimizations

- Inline SVG icons (no external requests)
- Optimized image loading (when added)
- Minimal external dependencies
- CSS utility classes (Tailwind)
- Dark mode with CSS variables

## Dark Mode Support

- Full dark mode implementation
- Automatic system preference detection
- Smooth color transitions
- Emerald theme maintained in dark mode
- Proper contrast ratios
